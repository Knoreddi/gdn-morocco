# GDN Morocco Project

This repository contains the foundational structure and baseline documentation for the GDN Morocco project.

## Repository Structure

- `README.md` — Overview and quick start.
- `PROJECT_RULES.md` — Working agreements and quality rules.
- `LESSONS_LEARNED.md` — Ongoing retrospective notes.
- `docs/radiasync-blueprint.md` — Architecture and delivery blueprint.
- `config/config.example.json` — Example runtime configuration.
- `flows/main-flow.json` — Main workflow definition.
- `scripts/` — Automation scripts (placeholder directory).
- `tests/` — Test artifacts (placeholder directory).

## Quick Start

1. Copy `config/config.example.json` to a local config file as needed.
2. Review `PROJECT_RULES.md` before making changes.
3. Use `flows/main-flow.json` as the primary orchestration template.
